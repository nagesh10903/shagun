Tech Stack

Backend: Python + FastAPI
Database: MySQL

Authentication: OTP-based mobile login
Payments: Razorpay / Stripe / Cashfree (depending on country)
Notifications: SMS + WhatsApp + Email
1. Business Overview

Shagun is a collaborative wedding gifting platform where a groom, bride, or host creates a wedding event and publishes a list of desired gifts with estimated costs.

Invitees receive a personalized invitation link and can:

View gift items
Contribute cash toward one or more items
Track contribution progress
Make online payments
Leave messages for the couple

Once the required amount for a gift is fully funded, further contributions to that gift are blocked.

9. Important Business Rules
Rule 1

Gift cannot receive more money than its cost.

remaining = gift.cost - gift.contributed_amount

if contribution > remaining:
    raise Exception("Contribution exceeds remaining amount")
Rule 2

Use database transaction locking to prevent race conditions.

Example:

SELECT * FROM gifts
WHERE id = ?
FOR UPDATE;
Rule 3

Only successful payments increase contribution amount.

Rule 4

Cancelled payments do not reserve gift funding.

Rule 5

When funded:

if contributed_amount >= cost:
    status = "FUNDED"
